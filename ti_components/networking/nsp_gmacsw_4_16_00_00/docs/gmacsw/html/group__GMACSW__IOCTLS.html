typedef struct CPDMA_OPEN_RX_CHANNEL_IOCTL_CMD_ CPDMA_OpenRxChannelIoctlCmd

Param struct for the Rx channel open IOCTL.

The Param struct for the Rx channel open IOCTL, containing a pointer to the channel config structure and a pointer to a channel handle variable, which will be filled in by the IOCTL. The returned channel handle is an opaque variable used to access the channel in other IOCTL commands.
